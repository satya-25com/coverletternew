# LetterForge AI

**AI-Powered Job Application Assistant** — Create cover letters, analyze job matches, generate interview questions, and optimize your job search — 100% offline with no APIs required.

## Features

### Core Tools

1. **Smart Cover Letter Generator**
   - Corporate, Modern, and Minimal templates
   - ATS optimization scoring
   - Live preview with PDF export

2. **Mock Recruiter Review Engine**
   - Analyzes cover letter quality (0-100 score)
   - Tone, grammar, readability analysis
   - Action verb detection
   - Improvement suggestions

3. **Interview Question Generator**
   - Technical, behavioral, HR, and scenario questions
   - Role-specific questions based on skills
   - Customizable difficulty levels
   - Download as text file

4. **Job Match Analyzer**
   - Compare resume skills against job descriptions
   - Skill gap analysis
   - Radar chart visualization
   - Readiness score

5. **Success Probability Predictor**
   - Weighted algorithm (Resume 25%, Job Match 25%, Cover Letter 20%, ATS 15%, Experience 10%, Skills 5%)
   - Interview chance prediction
   - Key success factors identification

6. **Career Coach Engine**
   - Personalized career roadmaps
   - Missing skills identification
   - Certification recommendations
   - Salary estimation

7. **LinkedIn Profile Analyzer**
   - Profile strength scoring
   - Keyword density analysis
   - Optimization suggestions

8. **GitHub Profile Analyzer**
   - Portfolio quality score
   - Technology stack analysis
   - Project quality assessment

9. **Message Generators**
   - Networking messages (LinkedIn connection, recruiter outreach, referral requests)
   - Thank you emails (post-interview, recruiter, HR)
   - Salary negotiation emails

10. **Application Strategy Generator**
    - Weekly application plans
    - Industry recommendations
    - Company targeting

## Dashboard

The Bento Grid Dashboard provides:
- KPI cards (ATS Score, Recruiter Score, Job Match, Success Rate)
- Skill gap tracker with radar charts
- Application funnel visualization
- Activity timeline
- Profile scores (LinkedIn, GitHub, Portfolio)

### Responsive Design

- **Desktop**: Bento Grid layout with interactive cards
- **Mobile**: Accordion layout with smooth transitions
- **State Persistence**: Active feature preserved across layout changes

## Technology Stack

- **Pure HTML/CSS/JavaScript** — No frameworks
- **Vanilla Canvas API** — Custom charts
- **IndexedDB** — Local data storage
- **LocalStorage** — Session preferences
- **No external APIs** — All intelligence is local

## Intelligent Analysis

The platform uses rule-based intelligence:
- **TF-IDF** for keyword extraction
- **String similarity** (Levenshtein, Jaccard, N-gram)
- **Readability scoring** (Flesch Reading Ease)
- **Grammar pattern detection**
- **ATS keyword matching**

## Getting Started

1. Open the application in a browser
2. Navigate using the top menu
3. Fill in forms and generate content
4. All data stays in your browser

## Privacy

- 100% offline functionality
- No data sent to external servers
- No cookies or tracking
- Data stored locally in your browser
